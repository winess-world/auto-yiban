># yiban_自动化工具

这是一个易班的自动化工具

```
--功能
 |-多账号刷易班EGAP自动脚本，班级日提高40+
```
>## requests版本
#### 目录结构

```
--requests_root
 |-cope_image.py // 图像处理模块（利用自己研究的算法对图像进行处理）
 |-orc_scheduling.py // ORC集中调度模块（集中了4个先进的ORC识别接口，并自动进行调度）
 |-login_yiban.py // 易班的自动登录模块（实现多账号密码自动登录）
 |-template.py // 元类模板模块 （对类的管理）
 |-package_all.py // 项目所用的所有包
 |-test.py // 模拟测试环境进行测试
 |-setting.pu //配置文件（可选择多种运行模式）
 |-ascension_yiban_pga.py //自动点赞，评论等
 |-main.py // 可实现批量化运行的入口
 

```
